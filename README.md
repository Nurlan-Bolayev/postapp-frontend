# Post App (frontend)

---

## Requirements

- nodejs 14 (others may work as well)
- Prepared backend running at (http://localhost:8000)

## Steps

1. Clone the project and `cd there`
2. Run `npm install`
3. Run `npm run serve`
4. Open http://localhost:8080
5. Go to login page
